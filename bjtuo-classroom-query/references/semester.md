# 学期参数参考

## 学期代码格式

学期代码格式：`YYYY-YYYY-X-Y`

- `YYYY-YYYY`: 学年范围（如 2025-2026）
- `X`: 学期序号（1为第一学期，2为第二学期）
- `Y`: 固定值 2

## 常用学期

| 代码 | 说明 |
|------|------|
| 2025-2026-1-2 | 2025-2026学年第一学期 |
| 2025-2026-2-2 | 2025-2026学年第二学期 |
| 2024-2025-1-2 | 2024-2025学年第一学期 |
| 2024-2025-2-2 | 2024-2025学年第二学期 |

## 使用示例

```bash
# 查询2025-2026学年第一学期的课表
python scripts/query_classroom.py --semester 2025-2026-1-2 --week 13
```

注意：学期代码必须与实际查询的周次匹配。
