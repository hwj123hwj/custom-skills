## 周次 (Weeks)

通常支持输入数字（如 "12"）或完整格式（如 "第12周"）。

| 周次 | 内部代码 (Value) |
| :--- | :--- |
| 第1周 | 1 |
| 第2周 | 2 |
| ... | ... |
| 第30周 | 30 |

## 学期 (Semesters)

默认选择当前学期，支持如下格式：

- 2026-2027-2
- 2026-2027-1
- ...
