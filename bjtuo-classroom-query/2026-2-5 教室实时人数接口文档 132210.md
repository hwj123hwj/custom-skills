# 教室实时人数接口文档

## 接口概述

本接口用于获取北京交通大学各教学楼教室的实时人数和使用情况。

## 接口地址

```
GET http://yaya.csoci.com:2333/api/classnum/?building={教学楼名称}
```

> 📌 **说明**：这是一个第三方提供的接口服务，返回 **JSON 格式数据**，而非文件。

## 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| building | String | 是 | 教学楼名称，需进行URL编码 |

```javascript
// 使用 fetch API（自动处理 URL 编码）
async function getClassroomCapacity(buildingName) {
  // ✅ 使用 encodeURIComponent 自动进行 URL 编码
  const url = `http://yaya.csoci.com:2333/api/classnum/?building=${encodeURIComponent(buildingName)}`;
  const response = await fetch(url);
  const data = await response.json();
  return data;
}
```


### 数据解析逻辑（来自源码）

根据 `ClassroomCapacityService.java:28-42` 的代码分析：

```java
// 1. 解析响应体为 JSON 对象
JSONObject jsonObject = new JSONObject(responseBody);

// 2. 提取时间范围（有效期）
JSONArray timeArray = jsonObject.getJSONArray("time");
String effectiveDateStart = timeArray.getString(0);  // 开始日期

// 3. 提取教室数据列表
JSONArray dataArray = jsonObject.getJSONArray("data");

    JSONArray classroom = dataArray.getJSONArray(i);
    String roomName = classroom.getString(0);   // 教室名称

```

### 返回字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| time | Array | 数据有效期范围 [开始时间, 结束时间] |
| data | Array | 教室数据列表 |
| data[i][0] | String | 教室名称（如 SY101） |
| data[i][1] | Float | 占用率百分比（如 5.55 表示 5.55%） |
| data[i][2] | Integer | 当前使用人数 |
| data[i][3] | Integer | 教室总容量 |

## 注意事项

1. **⚠️ URL编码是必须的**：中文参数必须进行URL编码，否则会返回 500 错误。大多数编程语言的HTTP库会自动处理，但使用curl时需手动编码。
2. **教学楼名称必须准确**：参数值必须与上述列表中的名称完全一致，包括"教学楼"、"楼"等后缀
3. **实时性**：数据反映的是当前时间点的教室使用情况，时间戳精确到秒
4. **有效期**：返回的数据包含时间范围，表示数据采集的时间段（通常相差几秒到十几秒）
5. **特殊值处理**：当 `Used >= Capacity` 时，表示无法正常读取该教室人数数据
6. **浏览器行为**：在浏览器中直接访问该 URL 可能会触发文件下载，这是正常的（因为响应头 Content-Type 可能是 `application/octet-stream`），实际内容是 JSON 文本
7. **服务状态**：⚠️ 该接口为第三方提供的服务（`yaya.csoci.com:2333`），非学校官方接口
8. **数据更新频率**：根据测试，数据大约每10-20秒更新一次

## 错误处理

### HTTP 状态码
- **200**：请求成功，返回 JSON 数据
- **500**：服务器内部错误，通常是因为中文参数未进行 URL 编码
- **其他**：请求失败，检查网络或参数

### 常见错误及解决方法

#### 错误1：500 Internal Server Error
**原因**：中文参数未进行 URL 编码
```bash
# ❌ 错误示例（直接使用中文）
curl "http://yaya.csoci.com:2333/api/classnum/?building=思源楼"
# 返回：500 Internal Server Error

# ✅ 正确示例（使用 URL 编码）
curl "http://yaya.csoci.com:2333/api/classnum/?building=%E6%80%9D%E6%BA%90%E6%A5%BC"
# 返回：JSON 数据
```

#### 错误2：无数据返回
**原因**：教学楼名称不正确或不支持
```bash
# 检查返回的 data 数组是否为空
```

### 异常判断
- **数据异常**：当 `Used >= Capacity` 时，通常表示传感器故障或数据读取异常，前端应提示"无法读取本教室人数"
- **服务不可用**：若持续返回错误，表示第三方服务暂时不可用，建议稍后重试

### 浏览器访问提示
在浏览器中直接访问该 URL 时，可能会触发文件下载行为。这是正常的，因为服务器可能设置了：
```
Content-Type: application/octet-stream
```
或
```
Content-Disposition: attachment
```
下载的文件内容实际上是纯文本的 JSON 数据，可以用文本编辑器打开查看。
