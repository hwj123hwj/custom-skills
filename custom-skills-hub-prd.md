# Custom Skills Hub - 产品需求文档 (PRD)

## 1. 产品概述
**Custom Skills Hub** 是一个专门用于展示和分发 GitHub 仓库 `hwj123hwj/custom-skills` 中 AI Agent 技能的个人网站。该网站旨在以现代化、开发者友好的方式呈现技能集合，灵感来源于 `skills.sh`，为用户提供清晰的技能描述、使用场景和安装指南。

### 1.1 目标用户
- **AI 开发者/爱好者**: 寻找现成 AI 技能以增强其 Agent 能力的人。
- **仓库维护者**: 需要一个美观的展示面来分享和推广自己的技能。

### 1.3 部署与同步 (方案 A: Mono-repo)
- **部署平台**: 腾讯云 EdgeOne Pages。
- **目录结构**: 网站源码存放在 `hwj123hwj/custom-skills` 仓库的 `/web` 目录下。
- **同步机制**: 采用 "构建时提取" 方式。每当仓库有任何提交（包括技能逻辑更新或网站代码更新），EdgeOne Pages 都会自动触发构建。构建过程中，脚本会自动扫描并解析仓库中的所有技能文档。

### 1.4 SEO 与分享优化
- **Meta 标签**: 每个技能详情页应有动态的 Title 和 Description，方便 SEO 收录。
- **Open Graph**: 支持社交媒体分享时的卡片预览 (OG Image)，最好能动态生成包含技能名称的图片。

## 2. 功能需求

### 2.1 技能展示 (Skill Catalog)
- **搜索与过滤**: 
  - 支持按技能名称或描述关键词搜索。
  - 支持按 Tag (如 Analysis, Crawler) 筛选。
- **卡片式列表**: 在首页以卡片形式展示所有技能。
- **关键信息**: 每个 card 应包含：
  - 技能名称 (如 `analyze-up`)
  - 简短描述 (一句话介绍功能)
  - 类别/标签 (如 `Analysis`, `Crawler`, `Utility`)
  - 对应的 Emoji 图标

### 2.2 技能详情 (Skill Details)
- **详细视图**: 点击卡片进入详情页或展开模态框。
- **详细内容**:
  - **名称与图标**
  - **完整描述**: 详细解释技能的功能。
  - **使用场景**: 列出具体的应用案例 (如 "总结某 UP 主观点")。
  - **安装命令**: 
    - 提供 **Skills CLI** 命令 (推荐)：`npx skills add ...`
  - **参数说明**: (可选) 简要说明支持的参数。

### 2.3 导航与链接
- **GitHub 链接**: 页面显著位置 (Header/Footer) 提供跳转到 `hwj123hwj/custom-skills` 的链接。
- **关于/帮助**: 简要介绍如何配置环境 (如 Python, uv, PostgreSQL)。

### 2.4 响应式设计
- 适配桌面端、平板和移动端设备。
- 移动端采用单列布局，桌面端采用网格布局。

## 3. 非功能需求
- **性能**: 页面加载速度快，静态资源优化。
- **可维护性**: 技能数据应与代码分离 (例如使用 JSON/TS 文件存储)，方便后续从 GitHub 自动同步或手动更新。
- **UI 风格**: 极简主义，黑色/深色主题 (Dark Mode)，强调代码感和科技感。

## 4. 内容清单 (初始数据)
需包含以下技能 (基于 GitHub 仓库信息):
1.  **analyze-up**: B 站 UP 主画像分析。
2.  **ask-kb**: 视频知识库问答。
3.  **build-kb**: 构建视频知识库向量索引。
4.  **crawl-and-export**: 采集视频并导出文稿。
5.  **download-video**: 下载视频并合并。
6.  **idea-incubator**: 产品孵化与方案制定.
7.  **skill-browser-crawl**: 浏览器网页爬取与 Markdown 提取。
