这份 recipe 用来从仓库里已经沉淀下来的 `knowledge-to-deck` 设计文档中，
提炼出最值得展示的两层内容：

- 为什么这条流水线值得存在
- 它的最小闭环到底是什么

目标不是做“全部说明书”，而是生成一份适合放到 Web 展示页的高密度流程型 deck。
