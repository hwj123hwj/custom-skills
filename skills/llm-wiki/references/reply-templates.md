```

知识被预先编译成结构化页面，而非每次提问都从零检索。

**现在请把源文件发给我：**

• 网页链接（直接发URL）

发过来后我来整理进知识库，可以一次发多个。
```

## HELP_MESSAGE

```
📚 知识库使用指南

**建立知识库**
说「建立知识库」或「帮我建个知识库」

**存入内容**
• 「把这个存进知识库」+ 链接/文字/文件
• 「记住这些内容」+ 粘贴内容
• 「帮我学习这篇文章」+ URL

**查询知识库**
• 「用我的知识库回答：[问题]」
• 「从wiki里找找关于[主题]的内容」

**管理**
• 「看看知识库里有什么」→ 状态概览
• 「整理一下知识库」→ 健康检查
• 「查看操作日志」→ 完整日志

⚡ 知识只从整理过的页面里查，存进来才能查到
```
