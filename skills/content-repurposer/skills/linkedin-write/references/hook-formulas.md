# Hook Formulas

Distilled hook patterns. Use the **shape**, never the branded names. Each pattern below shows the structural pattern and a real example from Anuj's writing or how it would adapt.

## 1. Dichotomy

Two parallel short statements that frame a tension the post will resolve.

> "Cognitive surrender costs you skill. Moral surrender costs you the architecture."

Works for: leadership, judgment, two-modes-of-failure topics.

## 2. Demoting

Reframe an over-hyped idea as smaller than people think.

> "Treat it as a strong default chunk-level upgrade, not a strategy."

Works for: pushing back on hype, contextual retrieval, fine-tuning, latest-model takes.

## 3. Pragmatic-flex

Lead with a specific number + ship outcome that contradicts the conventional choice.

> "The best infrastructure decision is the one that costs 40% less and ships today."

Works for: cost/latency/throughput posts, infrastructure choices, model selection.

## 4. Observation-flip

State an observation. Flip it into the uncomfortable implication.

> "Same logo. Same price. Different soul."

Works for: model drift, AI shrinkflation, silent regressions, vendor behavior changes.

## 5. Forensic opener

For security/incident topics. Name what happened, what was at risk, in one or two sentences. No drama.

> "The MCP server published last Tuesday loaded code from an attacker-controlled URL on every tool call. Anyone running it for the last 9 days is compromised."

Works for: CVE write-ups, supply chain, RCE, breach analysis.

## 6. Curiosity-gap (use sparingly)

Pose a question whose answer is non-obvious and is what the post delivers. Do NOT use this if the answer is obvious or if the question is generic.

> "Why does the same prompt give different results in batch mode than in interactive mode — even at temperature 0?"

Works for: debugging stories, agent behavior, edge cases.

## 7. Year-pivot

Use sparingly and only when there's a real shift this year that frames the post.

> "In 2026, the agentic stack stopped being a benchmark race and started being a context-window race."

Works for: trend pieces. Avoid this pattern if the post isn't genuinely tied to a timeframe.

---

## Patterns to NEVER use

- "Hot take:" / "Unpopular opinion:" → tells, not shows.
- "Most people get this wrong..." → cliché.
- "After X years in tech..." → credentialing as opener.
- Any opening that names what the post is about ("In this post, I'll...").
- Questions where the answer is obviously "yes" or where the question has been asked to death.
