## 一、TextContent Analysis

### 1.1 叙事结构分析（Stage 7）

按时间线拆解，**输出必须包含时长和占比**：

| 结构层 | 实际内容摘要 | 实际时长 | 占比 |
|--------|-------------|---------|------|
| 开场钩子 | [摘要] | Xs | X% |
| 核心卖点 | [摘要] | Xs | X% |
| 产品价值 | [摘要] | Xs | X% |
| 场景延伸 | [摘要] | Xs | X% |
| 行动号召 | [摘要] | Xs | X% |
| **合计** | - | **总时长s** | **100%** |

参考占比要求（基于90s视频等比缩放）：
- 开场钩子：控制在5s内
- 核心卖点（5-25s）+ 产品价值（25-65s）合计 ≥ 60%
- 行动号召：控制在10s内

### 1.2 叙事声音分析
- **人称视角**：第一/第二/第三人称
- **语气特征**：正式/口语/专业/亲切
- **节奏特点**：句子平均时长、语速变化

### 1.3 修辞手法识别

| 手法 | 说明 |
|------|------|
| 类比 | 抽象概念具象化，降低理解门槛 |
| 数据论证 | 具体数字，增强可信度 |
| 对偶 | 对仗工整，朗朗上口 |
| 呼应 | 首尾呼应，结构闭环 |
| 排比 | 并列结构，增强气势 |

### 1.4 词库提取

分类提取：核心词 / 场景词 / 卖点词 / 行动词

## 二、Viral-Abstract-Script 框架

### 2.1 Viral-5D 诊断

| 维度 | 说明 | 评分标准 |
|------|------|----------|
| **Hook** | 黄金3秒开头 | 是否立即抓住注意力 |
| **Emotion** | 情感触发 | 是否引发共鸣/笑点/惊讶 |
| **爆点结构** | 节奏设计 | 是否有高潮点 |
| **CTA** | 行动引导 | 是否有明确下一步 |
| **社交货币** | 传播价值 | 是否值得分享/讨论 |

### 2.2 风格定位

| 风格 | 特点 | 适合场景 |
|------|------|----------|
| 抽象鬼畜 | 意义消解、荒诞 | 谐音梗、整活 |
| 反转剧情 | 悬念→意外结局 | 故事类 |
| 共鸣治愈 | 真实感、普通人视角 | 生活记录 |
| 科技种草 | 理性、信息密度高 | 产品测评 |

### 2.3 Hook 类型参考

1. **痛点切入**："你是不是也遇到过..."
2. **反常识**："90%的人都不知道..."
3. **数字开头**："3个方法让你..."
4. **悬念设置**："结局没想到..."
5. **情绪共鸣**："太真实了..."

## 三、执行迭代框架

### 3.1 脚本结构
- **整体结构**：线性叙事 / 问答式 / 对比式 / 悬念式
- **段落逻辑**：各段衔接方式和转场节奏
- **内容密度**：信息量分布是否均匀

### 3.2 内容亮点
- **金句**：最有传播力的1-3句话，直接摘录
- **独特视角**：别人没说过的角度
- **情绪峰值**：引发最强情绪反应的节点

### 3.3 优化建议

给出3条具体可执行的改进方向（"问题→建议→预期效果"），优先针对 Viral-5D 评分最低的维度。
