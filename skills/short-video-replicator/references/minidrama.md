# 小剧情脚本生成指南

> 生成3个差异化的小剧情脚本版本（Step 5）
> **去AI化全程优先** — 参照 `references/de-ai-guide.md`
> 最重要的一件事：台词要像真人说话。写完每句台词问自己：**这句话，我会对朋友说吗？**

---

## 完整结构（4段，缺一不可）

### 第一段：痛点剧情（10-30s）
**目标**：让观众在10秒内说出"这就是我"。

剧情要丰富，不是几句话带过。必须包含：
- 具体场景（不是"在家"，是"凌晨两点，盯着电脑屏幕"）
- 真实细节（不是"很累"，是"第三次改方案，客户还是说不满意"）
- 情绪变化（从正常 → 崩溃/愤怒/绝望，要有过程）
- 角色互动（两人对话比独白更有张力，矛盾要具体）

冲突类型（自由发挥）：被误解越解释越说不清 / 明明努力了结果一团糟 / 以为占便宜其实踩坑 / 熟悉的人突然说了句戳心的话

### 第二段：转场（5s左右）
从剧情自然切换，不能硬切广告：
- 角色内心独白："那一刻我才想到……"
- 快进字幕："后来我发现了一个方法"
- 旁白直接点破："其实这个问题，很多人都遇到过"

### 第三段：产品价值（20-40s）
必须展开写三件事（不能一带而过）：
- **解决什么**：对应剧情痛点，直接回应
- **为什么是它**：和其他方法相比具体差在哪
- **用了之后**：生活/工作有什么具体变化，越具体越好

### 第四段：行动号召（5-10s）
自然收尾，接着剧情情绪走，带一点紧迫感或情感钩子，不要硬推销。

---

## 版本差异化

3个版本选3种**完全不同**的痛点场景：

| 方向 | 剧情风格 | 核心情绪 |
|------|---------|---------|
| 情感共鸣 | 日常真实场景，细节打动人 | 感同身受、心疼 |
| 反转惊喜 | 以为会怎样，结果完全不同 | 惊讶、意外、好奇 |
| 压力释放 | 被逼到墙角，意外找到出路 | 窒息感→解脱感 |

## 镜头节奏参考

| 段落 | 建议时长 | 景别风格 |
|------|---------|---------|
| 痛点剧情 | 10-30s | 近景/特写，跟拍，有临场感 |
| 转场 | 5s | 字幕/快切/定格 |
| 产品价值 | 20-40s | 中景+产品特写，干净清晰 |
| 行动号召 | 5-10s | 近景，眼神直视镜头，产品定格 |

```
### 镜头建议

| 产品价值（Xs-Xs） | [中景+口播/展示] |
| 行动号召（最后5-10s） | [近景+产品定格] |

【痛点剧情】

【场景】[具体地点+时间+氛围，一句话]

（场景动作）
角色A：「台词——要有口语感，有情绪」
角色B：「台词——矛盾要具体，不要泛泛」
...（剧情要有起伏，有转折，不能平铺直叙）

【转场】
[角色内心独白 / 字幕 / 旁白，自然切换]

【产品价值】
[口播或对话，包含：解决什么 → 为什么是它 → 用了之后的具体变化]

【行动号召】
「[自然CTA，接着剧情情绪走]」
字幕：[CTA文字]

**创作说明**: [痛点逻辑 + 产品植入方式 + 与其他版本差异]
```

## 注意

- 痛点剧情要丰富，有细节有情绪，**不是几句话带过**
- 3个版本痛点场景**完全不同**，让用户有真实选择
- 台词字数 300-800字（约60-160s），动作描述不计入
- **去AI化优先于一切格式要求**
